# 🐆 PantherSS — iOS Privacy Scanner

**Scanner Anti-Cheat Avançado para Free Fire (iOS)**

---

## 📌 Sobre o Projeto

**PantherSS** é um scanner avançado para dispositivos iOS que analisa o **Relatório de Privacidade** do iPhone, identificando de forma rápida e eficiente proxies, cheats, sideload, jailbreak e infraestruturas suspeitas utilizadas por cheaters no Free Fire.

O projeto tem como base o **KellerSS** (criado por KellerSS) e foi **aprimorado por Yakashi**, da **Organização Panther**, com o objetivo de oferecer uma ferramenta mais completa e visual para a comunidade.

> **"Pensado e realizado em prol da comunidade de Free Fire, sem fins lucrativos."**

## ✨ Melhorias do PantherSS

Esta versão foi aprimorada por **Yakashi (Org Panther)** e conta com:

- Interface moderna e profissional (estilo AMOLED premium)

## 🚀 Como Utilizar

### Passo 1 — Instale o Scriptable
Baixe o aplicativo **Scriptable** na App Store.

### Passo 2 — Exporte os relatórios

1. **Relatório de Privacidade:**
   - Ajustes → Privacidade e Segurança → Relatório de Privacidade de Apps → Exportar

2. **Dados de Análise (Recomendado):**
   - Ajustes → Privacidade e Segurança → Análise e Melhorias → Exporte o arquivo mais recente (`xp_amp_app_usage...`)

### Passo 3 — Instale o PantherSS
- Abra o Scriptable → Novo Script → Cole todo o código → Salve como `PantherSS`

### Passo 4 — Execute
Toque no script, selecione os arquivos e aguarde a análise completa.

## 🎯 O que ele detecta?

- Apps de Proxy / VPN / MITM (Potatso, Shadowrocket, Surge, etc.)
- Jailbreak e ferramentas de sideload (TrollStore, Dopamine, ESign, etc.)

- Infraestrutura de cheats conhecidos
- VPS / Hosting / Proxy (Hostinger, Multacom, OVH, DigitalOcean...)
- Apps Fantasma e atividade suspeita
- Picos de tráfego durante partidas
